# Fonts — attribution

This kit self-hosts the following typefaces. Verify each license before redistributing.

- **IBM Plex Sans Arabic** — © IBM, released under the SIL Open Font License 1.1 (OFL).
  Free to use, bundle, and redistribute with the license notice.
- **Thmanyah Sans** / **Thmanyah Serif** — type published by Thmanyah at
  <https://font.thmanyah.com>. Bundled here for demonstration; confirm Thmanyah's
  license terms permit redistribution before using in a public/production project.
  If redistribution is not permitted, reference the fonts from their source instead of
  committing the `.woff2` files.
